Quickstart (NL)
1 Installeer Ollama
   Volg de instructies op de Ollama‑website en zorg dat het ollama‑commando werkt in je terminal.

2 Download het Mistral‑model

   bash
   ollama run mistral
   Wacht tot het model één keer is gedownload en gestart; daarna kun je dit commando afsluiten.

3 Start de PDA‑agent in de terminal

   bash
   cd /pad/naar/PDA_LLM
   python main.py

4 Chatten
Na het starten van main.py kun je rechtstreeks in de terminal met de PDA‑chatbot praten.
